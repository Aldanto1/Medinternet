# Документация проекта «Мединтернет»

| Файл | Что внутри |
|---|---|
| [`DEVELOPERS.md`](DEVELOPERS.md) | **Основная документация проекта**: архитектура, структура репозитория, переменные окружения, схема БД и роли, ключевые потоки (регистрация, навигация в боте, ИИ-поиск, CRM-рассылки), локальный запуск, деплой на Railway, ручные операции и грабли. |
| [`RXCODE_AI_API.md`](RXCODE_AI_API.md) | Справочник по API нейросети **RX Code AI** в читаемом виде: подключение и авторизация, эндпоинты, формат стрима, замеренное поведение, неиспользуемые возможности. |
| [`swagger-mi.json`](swagger-mi.json) | Официальная спецификация RX Code AI API (Swagger 2.0) — исходный файл от разработчиков нейросети. |

> Секреты (токены, строки подключения, ключи API) в документацию **не попадают** —
> они только в `env/.env` (в `.gitignore`) и в переменных Railway.
